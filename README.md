# GBEmulator

A Gameboy emulator in Java with the GUI in Swing

## What has been implemented
* CPU Emulation
* MBC1, ROM and ROM+RAM cartridge
* Background
* Sprites
* Interrupts
* Instruction Timing
* Timer and Divider
* DMA transfer
* Joypad

## What is not yet complete
* Window
* LCD, Window, Sprites enable / disable
* Serial communication
* Sound
* Extensive testing
