# GBEmulator

A Gameboy emulator in Java with Swing (incomplete and barely tested)

## What has been implemented
* CPU Emulation
* MBC1, ROM and ROM+RAM cartridge
* Background
* Interrupts
* Instruction Timing and Timers

## What is not yet complete
* Window and Sprites
* Joypad input
* Serial communication
* Sound
* Extensive testing
